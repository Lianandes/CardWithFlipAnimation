# Card With Flip Animation

Projeto desenvolvido para praticar animações e efeitos visuais utilizando HTML e CSS através da criação de um card com efeito flip 3D.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em animações CSS, transformações 3D e construção de componentes interativos para interfaces web modernas.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Efeito flip animation
- Transformações 3D
- Hover interativo
- Design moderno
- Transições suaves

## 📚 Conceitos praticados

- CSS Transform
- RotateY
- Perspective
- CSS Transitions
- Efeitos interativos
